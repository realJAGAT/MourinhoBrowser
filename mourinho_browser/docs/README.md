# Mourinho Browser Documentation

This documentation covers the architecture, feature set, and developer setup for Mourinho Browser.

## Components

- `main.py`: Browser entry point.
- `browser/`: Contains all browser feature modules and UI integration.
- `database/`: SQLite schema and database initialization support.
- `ui/`: Qt Designer UI definitions.
- `themes/`: Style sheets for the browser themes.
- `assets/`: Logo and asset directories.
- `tests/`: Pytest unit tests for backend modules.
